# Introduction

Welcome to **davidlog.io** — my personal tech notebook.

## What's here

This site collects notes, patterns, and hard-won lessons from day-to-day engineering work. The focus areas are:

- **C# / .NET** — internals, performance, DI, EF Core tricks
- **Databases** — SQL Server, PostgreSQL query tuning, Redis patterns
- **DevOps** — Docker, Kubernetes, CI/CD, Linux tips
- **Angular** — signals, standalone components, reactive patterns

## How to use

Browse via the sidebar on the left. Each page is a living document — updated as I learn more or find better approaches.

## Tech stack of this site

- [Angular 21](https://angular.dev) — zoneless, standalone components
- [ngx-markdown](https://github.com/jfcere/ngx-markdown) — markdown rendering
- [Tailwind CSS v4](https://tailwindcss.com) — styling
