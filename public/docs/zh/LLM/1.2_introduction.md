# NLP 與 LLM 入門

## NLP

自然語言處理是語言學和機器學習的一個分支，專注於理解與人類語言相關的一切。

### 常見的自然語言處理工作

- **Classifying whole sentences 整句分類**
- **Classifying each word in a sentence 將句子中的每個字分類**
- **Generating text content 產生文字內容**
- **Extracting an answer from a text 從文本中提取答案**
- **Generating a new sentence from an input text 根據輸入文字產生新句子**

> 自然語言不限制文本，也能處理語音識別和電腦視覺領域的複雜挑戰。

### Transformer 不只做 NLP

Transformer 架構本來是為文字設計的，後來被拿去解決語音和視覺的問題：

**語音 — Whisper**
OpenAI 推出的語音辨識模型。它是 Encoder-Decoder 架構，在 680,000 小時的音頻資料上訓練。
它不需要針對特定語言 fine-tune，直接 zero-shot 就能辨識多種語言。
流程是：原始音頻 → Log-Mel 頻譜圖 → Encoder 編碼 → Decoder 自回歸輸出文字。

**視覺 — ViT（Vision Transformer）**
圖片沒辦法直接餵進 Transformer，所以 ViT 的做法是把圖片切成一塊塊小 patch（類似文字的 token），然後把這些 patch 當 token 一樣丟進 Transformer Encoder。
本來要處理像素，現在變成處理「圖片的詞彙」，概念上跟文字任務幾乎一樣。

### 語言處理具有挑戰性

因語言的複雜性，仍然存在許多根本性的挑戰，這些挑戰包括 **理解歧義**、**文化背景**、諷刺和幽默。
語言學習可以透過多樣化的資料集上進行大規模訓練，但在許多複雜情境中，其理解能力往往達不到人類水平。

## LLM

大語言模型(Large language model) 是基於海量的文本資料訓練的人工智慧模型。
能夠理解和產生與人類語言相似的文本，能識別各種語言模式，並執行各種語言工作，且不需要針對特定任務進行訓練。這也徹底改變了自然語言處理領域。像 GPT ( Generative Pre-trained Transformer)和 Llama 等架構改變了語言處理的可能性。

### 特點

- **規模** (Scale)：它們包含數百萬、數十億甚至數千億個參數
- **通用能力(General capabilities)** ：無需接受特定任務訓練即可執行多項任務
- **情境學習(In-context learning)** ：模型可以從提示中提供的例子學習
- **湧現能力(Emergent abilities)** ：隨著這些模型規模的擴大，模型展現出一些並非預先設定或預料到的能力

### 限制

- **幻覺(Hallucinations)** ：模型可以自信地產生錯誤訊息。
- **缺乏真正理解(Lack of true understanding)** ：他們缺乏對世界的真正理解，僅依據統計模式行事
- **偏差(Bias)** ：它們可能會重現訓練資料或輸入中存在的偏差
- **上下文視窗(Context Window)** ：模型的上下文視窗數量有限（不過這種情況正在改善）
- **運算資源(Computational resources)** ：模型需要大量的運算資源

## 偏見和局限性 (Bias and Limitations)

模型功能強大，但也有限制性，其限制性在於，為了能夠利用大量資料進行 Pretraining，通常研究人員會抓取所有能夠找到的內容。這包括了網路上最好的資料，也包括了最壞的資料。所以這些模型很容易產生帶有性別歧視、種族歧視等等的內容。即使對模型進行 Fine-tuning，也無法消除這種內在的 **偏見(Bias)**。

**具體的例子讓人很不舒服但很說明問題**

用 BERT 的 fill-mask 填入「工作職業」：

```python
# "This man works as a [MASK]."
# → lawyer, carpenter, doctor, waiter, mechanic

# "This woman works as a [MASK]."
# → nurse, waitress, teacher, maid, prostitute
```

注意：BERT 訓練資料是英文維基百科跟 BookCorpus，算是「相對中立」的資料來源，不是亂抓的網路垃圾——但偏見還是在。

這說明了一件事：**偏見不只來自爛資料，語言本身就帶有社會偏見，而模型學的就是語言的統計規律。** 所以 Fine-tuning 只能改變輸出行為，無法改變模型骨子裡學到的偏見。在正式場景使用這些模型時，這是必須時時留意的風險。
